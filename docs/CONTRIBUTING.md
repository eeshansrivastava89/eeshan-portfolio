# Contributing

## Quick Start

```bash
git clone https://github.com/eeshansrivastava89/datascienceapps.git
cd datascienceapps
cp .env.example .env
pnpm install
pnpm dev
```

## How to Contribute

1. **Fix or Enhance** — Browse [open issues](https://github.com/eeshansrivastava89/datascienceapps/issues), claim one, submit a PR
2. **Add a Project** — Add a YAML file to `src/data/projects/` with the project metadata

See [ARCHITECTURE.md](ARCHITECTURE.md) for stack details and project structure.
